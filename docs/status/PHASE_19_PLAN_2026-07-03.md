# Phase 19 Plan - Live-Data Artifact Readiness

Date: 2026-07-03

Phase 19 is a real-data artifact-readiness repair pass, not feature expansion. It uses the persisted FMP bars and quote snapshots from Phase 18 to make dirty pipeline state visible, rebuild derived artifacts in dependency order, and dry-run the research cycle with strict stale gates.

## Guardrails

- No broker execution, order routing, options workflow, production WebSocket dependency, black-box auto-learning loop, auto-activation, or profitability claim.
- Do not weaken stale gates. Strict research-cycle dry runs must use `allow_stale=false`; one diagnostic `allow_stale=true` run is allowed only if explicitly labeled.
- Do not call FMP during feature, candidate, label, or replay rebuilds. The rebuild source is persisted bars/features/candidates only.
- Do not commit runtime databases, raw FMP responses, exports, logs, or secrets. `FMP_API_KEY` remains runtime-only.

## Implementation Steps

1. Add exact dirty-window audit and exact-window clean marking for persisted `pipeline_build_windows`.
2. Add an artifact-readiness service that reports dirty windows plus persisted source summaries for bars, quote snapshots, freshness reports, replay runs, and research cycles.
3. Rebuild in dependency order: features, candidates, labels, replay runs, then freshness and research dry run.
4. Add API endpoints and scheduler job types for bounded local-only rebuild jobs.
5. Add focused tests for multi-session dirty-window repair, API visibility, and scheduler rebuild dispatch.
6. Run the local real-data dry run against Phase 18 persisted bars, export sanitized reports, update handoff/runbooks, then commit and push.

## Expected Outcomes

- Dirty-window audit is persisted/exportable and contains no secrets.
- Feature, candidate, label, and replay dirty windows for the selected scope are clean after rebuild.
- Strict research-cycle dry run either passes using historical-reference freshness or reports the exact blocker.
- Default wall-clock freshness may remain `STALE` if Phase 18 bars are historical relative to 2026-07-03; that is expected and must be documented separately from research-cycle scope readiness.
