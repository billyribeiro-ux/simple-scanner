## Bounded Defaults

- Entitlement symbols: `SPY,QQQ,AAPL,NVDA`.
- Seed symbols: `AMZN,AAPL,TSLA,SPY,QQQ,IWM,NVDA,GOOGL,BABA,SHOP`.
- Seed intervals: `1day,1min,5min,15min`.
- Max symbols: 10.
- Max intraday span: 5 days.
- Required reviewed endpoints for seed: quote batch if quotes are included, EOD full if daily bars are included, and the selected intraday endpoints if intraday bars are included.

## Non-Goals

No broker execution, order routing, option analytics, market internals, Level 2, dark-pool/order-book data, production WebSocket ingestion, black-box ML, automatic model activation, automatic deployment, self-learning claims, or profitability claims are in scope.
